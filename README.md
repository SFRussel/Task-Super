# TaskSuper

> The best simple task manager with drag & drop, real-time sync, and beautiful UI.

## Quick Start

```bash
# Install
npm install

# Configure Firebase
# Update src/services/firebase.js with your config

# Run
npm run dev
```

## Features

✅ Drag & drop tasks between lists  
✅ Mark tasks complete with checkbox  
✅ Filter by All/Active/Completed  
✅ Beautiful confirmation dialogs  
✅ Real-time Firebase sync  
✅ Custom backgrounds & profiles  

## Usage

- **Create Task**: Click "+ Add Task"
- **Complete Task**: Click checkbox
- **Move Task**: Drag between lists
- **Filter**: Use buttons at top
- **Delete List**: Click ⋮ → Delete List

## Tech Stack

React • Firebase • Vite • react-beautiful-dnd • Lucide Icons
